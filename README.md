宠物智能管理系统
基于 LVGL 的嵌入式宠物智能管理系统，适用于 Linux 帧缓冲设备（/dev/fb0），提供宠物状态监控、设备控制等功能。
项目信息
项目名称：宠物智能管理系统
作者：Johnson Li
制作时间：2025/10/14
版本号：v1.2
用途：宠物日常管理（喂食、环境监控等）的可视化控制终端
项目简介
本系统基于 LVGL（嵌入式 GUI 库）开发，运行在 Linux 系统的帧缓冲设备（/dev/fb0）上，提供图形化界面实现宠物相关设备的智能管理。支持宠物喂食器控制、环境温度 / 湿度监控、设备状态显示等功能，适合嵌入式 Linux 设备（如开发板）部署。
环境依赖
操作系统：Linux（支持帧缓冲设备/dev/fb0，如 Ubuntu、嵌入式 Linux 发行版）
编译工具：gcc、make
依赖库：libpthread（多线程支持）、LVGL（已通过子模块集成）
安装与编译
克隆仓库
bash
git clone <仓库地址>
cd <仓库目录>
初始化子模块（关键步骤，否则缺失 LVGL 核心组件）
bash
git submodule update --init --recursive
编译项目在当前目录执行make命令，自动编译生成可执行文件：
bash
make
运行方式
编译完成后，生成的可执行文件为demo，直接运行即可：
bash
./demo
注意：运行可能需要 root 权限（访问/dev/fb0设备），若提示权限不足，可使用sudo ./demo。
主要功能
主页：显示宠物状态概览（如最近喂食时间、环境温湿度）
喂食控制：远程控制喂食器，设置定时喂食计划
环境监控：实时显示宠物活动区域的温度、湿度数据
设备管理：查看喂食器、传感器等设备的在线状态
历史记录：查询喂食记录、环境数据变化趋势
注意事项
本项目仅供学习参考，实际部署需根据硬件设备（如传感器、执行器）进行适配
确保运行设备已启用帧缓冲（/dev/fb0存在），且支持触摸屏（如需交互操作）
若编译失败，检查子模块是否完整初始化，或依赖工具是否安装（如make、gcc）
